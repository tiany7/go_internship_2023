# 2023暑期实习gogogo！

## 项目介绍
本repo为“甜酱蜜城甜蜜蜜！”群群友共同项目，群友自发组织创建，旨在为即将到来的2023年暑期实习准备，为已经拿到牌子的群友们保驾护航，为退役老人们积累管理经验。争取在即将到来的2023年春招中做足准备。

本项目内容主要侧重点为后端内容，主要语言为cpp，后续考虑假如更多语言内容。学习项目之余会加入基础知识，八股文等学习。我们坚信，实践方才出真知。
